JavaScript method map(), filter(), forEach(), find() are the difference between them.

map(): map() method does something with each element of an array and returns a completely new array.

forEach(): forEach() method does something with an element like map().But map() return something and forEach() doesn't return anything.

filter(): A filter is performed on the array.A condition is passed to the filter method.Returns a completely new array by doing something with only those elements of the array for which the condition is true.

find(): method is also performed on the array like filter() method.And Passed a condition to the method. But find() method return only an element which element is fill up this find() condition.
